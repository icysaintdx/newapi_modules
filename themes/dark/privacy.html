隐私政策

Saint Code 非常重视用户的隐私保护。本隐私政策说明了我们在使用服务时如何收集、使用、存储和保护您的个人信息。

信息收集

  • API 使用数据：请求时间、模型选择、使用量统计
  • 账户信息：API Token、邮箱地址（可选）
  • 技术信息：IP地址、浏览器类型、访问日志

信息使用

我们收集的信息仅用于：

  • 提供和优化 AI API 服务
  • 处理账单和费用结算
  • 改善用户体验和平台性能
  • 发送重要的服务通知

数据安全

我们采用行业标准的安全措施保护您的数据，包括加密传输、访问控制和安全审计。
